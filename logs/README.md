# logs/

Эта папка создаётся автоматически при запуске `app.py`.

- Внутри появляются файлы `session_*.jsonl` с историей диалога и `usage` по токенам.
- **Реальные логи не коммитятся в публичный репозиторий**, потому что содержат пользовательские сообщения.

Если нужен пример формата — см. `sample_session.jsonl`.
